# Human 3.0 自我探索測評

An AI-powered self-discovery assessment based on Dan Koe's **Human 3.0** framework. Fill out a questionnaire across four life quadrants and receive a personalised development report.

## Features

- **19 deep questions** across Mind, Body, Spirit, and Vocation
- **Direct AI analysis** — paste your API key and get a report instantly on the page
- **Export Prompt** — no API key? Export a full prompt to paste into Gemini, Claude, or any AI
- **Download HTML report** — save your results as a beautiful dark-themed HTML file
- **Gemini Canvas tip** — instructions to have Gemini output a polished HTML report via Canvas
- **No backend** — single HTML file, all data stays local

## Usage

1. Open `human3-assessment.html` in any browser
2. Click **⚙️ 設定** to add your API key (optional)
3. Click **開始測評** and answer the questions
4. Receive your personalised **Human 3.0 Development Report**

### With API Key
The report is generated directly on the page. Download it as an HTML file.

### Without API Key
Export the full prompt and paste it into any AI. For a beautiful report, paste into [Gemini](https://gemini.google.com) and ask it to output an HTML file via Canvas.

## Framework

Human 3.0 maps development across **four quadrants** and **three levels**:

**Quadrants**
- **Mind** — beliefs, worldview, metacognition
- **Body** — health, energy, physical habits
- **Spirit** — relationships, meaning, community
- **Vocation** — career, value creation, mission

**Levels**
- **1.0 Conformist** — external authority, rule-following
- **2.0 Individualist** — self-directed, status-seeking
- **3.0 Synthesist** — integrative, creates new possibilities

> Based on the Human 3.0 framework by [Dan Koe](https://thedankoe.com)
